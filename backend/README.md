# Requisitos

### A aplicação web permitirá aos usuários listar, cadastrar, editar e excluir projetos, cada um contendo um título e uma descrição.

#### Entidades

- Usuário
- Projetos

#### Rotas

Usuário

- Criar Usuário
- Logar Usuário
- Editar Usuário

Projeto

- Criar projeto (titulo e descrição)
- Editar projeto
- Excluir projeto
- Listar projetos
